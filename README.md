# FinanceHub
A personal finance backend built using Spring Boot, PostgreSQL and Docker.


# FinanceHub

FinanceHub is a backend application that allows users to:

- Create an account
- Manage bank accounts
- Deposit money
- Withdraw money
- Transfer money
- Track expenses
- View analytics
- Set savings goals

Tech Stack

- Java
- Spring Boot
- PostgreSQL
- Maven
- Docker
